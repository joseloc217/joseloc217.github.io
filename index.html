Desarrollo Web Front-end

Es una especialidad para el desarrollo web que trabaja la interfaz web y hace que el usuario pueda interactuar con nuestra web. Está orientado a lenguaje de marcas y al lenguaje de programación web de ejecución en equipos clientes, sin necesidad de uso de servidores externos

Module Bundlers

Una herramienta que 'toma' piezas de javascript con sus dependencias y las empaqueta en un archivo para su uso

Preprocesadores

Permiten usar funcionalidades en CSS y JavaScript que tu navegador no admite de forma nativa

Lenguajes

Constituyen la base de su desarrollo. Sus principales lenguajes son HTML, CSS y javascript

Frameworks

Son proyectos que aglutinan bibliotecas de código y patrones de diseño avanzados, y sirven como base para el desarrollo de aplicaciones diversas
